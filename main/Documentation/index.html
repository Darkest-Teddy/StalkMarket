Thank you for purchasing Top-down Farming Game Kit. With this pack you will create an awesome Top-down Farming. With everything you need to get started, tiles, effects and characters made by a professional pixel artist. + some hud parts and touch controls

Files

Start by extracting your zip-file downloaded from Envato with any extraction software to a local folder. For MAC/OSX you can use UnrarX and for PC/Windows you can go with WinRAR.

Folders

All images are located in the folders where you extracted the zip-file, they are well structured to help you get started quicker. The images are all in .PNG and can be edited with any image software. The tiles have been prepared for easy import to UNITY, GMS2, DEFOLD, JSON & GODOT.

 

Animation

The animations have all been sliced into different images, named after the correct animation. 

File name cheatsheet

Example: tdfarmchar_0_idlelft_0.png

rgt = facing to the right

dwn = facing down

lft = facing to the left

up = facing up

tdf = game kit abbreviation

char_0 = character number

idlelft_0 = Frame number of animation. 0 is always the first frame.
